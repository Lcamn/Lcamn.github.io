一、 新建maven web 项目

  • 导包
  • 插件
    • 编译插件
    • war 编译插件
    • mybatis 反向生成器
  • 修改maven 资源文件路径(可选)
  • 新建对应的包结构

二、 配置spring mvc 和 IOC容器

  • 在web.xml 中配置前端控制器 DispatcherServlet

    • 读取spring-mvc配置文件，默认是 /WEB-INF/servlet名-servlet.xml, 一般不采用默认的方式

      • 指定spring-mvc.xml 位置
         <servlet>
       <servlet-name>mvc</servlet-name>
        <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
        <init-param>
          <param-name>contextConfigLocation</param-name>
          <param-value>classpath:/spring/spring-mvc.xml</param-value>
        </init-param>
   </servlet>
  <servlet-mapping> 
        <servlet-name>mvc</servlet-name>
        <url-pattern>/</url-pattern>
  </servlet-mapping>

三、编写 spring-mvc.xml 文件

  • 保证spring-mvc.xml 的文件头: 至少要包含三个名称空间 bean,context,mvc

  • 指定扫描的包路径,只认 @Controller 注解

    <context:component-scan base-package="net.suncaper.ssmdemo" use-default-filters="false">
      <context:include-filter type="annotation" expression=" org.springframework.stereotype.Controller"/>
</context:component-scan>

  • 添加注解驱动

    <mvc:annotation-driven></mvc:annotation-driven>

  • 忽略静态资源

    <!-- 方式1: 依赖服务器， default-servlet-name="default" 中default是服务器中静态servlet的名字，服务器有差异-->
 <mvc:default-servlet-handler default-servlet-name="default"/>
<!-- 方式2:: 不依赖服务器，配置有点麻烦-->

  • 配置视图解释器

    <bean id="internalResourceViewResolver"
    class="org.springframework.web.servlet.view.InternalResourceViewResolver">
<!--  <property name="viewClass" value="org.springframework.web.servlet.view.JstlView"/>-->
  <property name="prefix" value="/WEB-INF/views/"/>
  <property name="suffix" value=".jsp"></property>
</bean>

  • 配置文件上传解释器

    <bean id="multipartResolver" class="org.springframework.web.multipart.commons.CommonsMultipartResolver">
  <property name="defaultEncoding" value="UTF-8"></property>
  <property name="maxUploadSize" value="1024000"></property>
</bean>

四、 通过json 的方式测试spring mvc

              String[] jsonResult = new String[]{"中国","美国","泰国"};
              return jsonResult;
       }
   }

五、 整合IOC 容器

  • 新建spring-beans.xml

    <context:component-scan base-package="net.suncaper.ssmdemo">
  <context:exclude-filter type="annotation" expression="org.springframework.stereotype.Controller"/>
</context:component-scan>

  • 在web.xml 中通过 ContextLoaderListener 加载其它spring 配置文件

    <context-param>
  <param-name>contextConfigLocation</param-name>
  <param-value>classpath:/spring/spring-beans.xml</param-value>
</context-param>
<listener>
  <listener-class>org.springframework.web.context.ContextLoaderListener</listener-class>
</listener>

六、 生成对应Mybatis的文件

七、 整合Mybatis

  • 创建数据源

    <context:property-placeholder location="classpath:/mybatis_cfg/jdbc.properties"/>
<!-- 1. 连接数据库，创建数据源-->
<bean id="myDataSource" class="com.alibaba.druid.pool.DruidDataSource">
    <!--至少要配置 4 个基本元素 -->
   <property name="driverClassName" value="${db.driver}"/>
   <property name="url" value="${db.url}"/>
   <property name="username" value="${db.username}"/>
   <property name="password" value="${db.password}"/>
</bean>

  • 给数据源指定事务管理器

    <!--2. 创建事务管理器,指定给数据源 -->
<bean id="txManager" class="org.springframework.jdbc.datasource.DataSourceTransactionManager">
       <property name="dataSource" ref="myDataSource"/>
</bean>

  • 给事务管理器指定管理规则

    <!--3. 指定事务管理器的管理规则 ,通过注解的方式指定事务-->
<tx:annotation-driven transaction-manager="txManager"/>

  • 创建 SqlSessionFactory

    <bean class="org.mybatis.spring.SqlSessionFactoryBean" id="sqlSessionFactoryBean">
  <property name="dataSource" ref="myDataSource"/>
  <property name="mapperLocations" value="classpath:net/suncaper/ssmdemo/mapper/*.xml"></property>
  <property name="typeAliasesPackage" value="net.suncaper.ssmdemo.common.domain" />

    <!--A: 加载 mybatis-config.xml 配置文件 -->
   <property name="configLocation" value="classpath:/mybatis_cfg/mybatis-config.xml"></property>
   <!-- B: 创建Configuration对象-->
   <property name="configuration">
       <bean class="org.apache.ibatis.session.Configuration">
           <property name="lazyLoadingEnabled" value="true"/>
       </bean>
   </property>
<!--A  和B 只  能取其一-->
</bean>

  • 配置 MapperScanner,找到Mapper接口位置

    <bean class="org.mybatis.spring.mapper.MapperScannerConfigurer">
   <property name="basePackage" value="net.suncaper.ssmdemo.mapper"/>
   <property name="sqlSessionFactoryBeanName" value="sqlSessionFactoryBean"/>
</bean>

十、处理编码

以下三个条件基本可以避免出现乱码问题

  • 添加编码过滤器

       <filter>
          <filter-name>CharacterEncodingFilter</filter-name>
          <filter-class>org.springframework.web.filter.CharacterEncodingFilter</filter-class>
          <init-param>
              <param-name>encoding</param-name>
              <param-value>UTF-8</param-value>
          </init-param>
          <init-param>
              <param-name>forceRequestEncoding</param-name>
              <param-value>true</param-value>
          </init-param>
      </filter>

  • 数据库建库语句和建表语句，尽量加上编码

      create  database mybatisdemo charset = utf8;
  use mybatisdemo;
  DROP TABLE IF EXISTS `employee`;
  CREATE TABLE `employee`(
    `id`  int NOT NULL AUTO_INCREMENT,
    `name`  varchar(32) NOT NULL COMMENT '员工名称',
    PRIMARY KEY (`id`)
  ) ENGINE=InnoDB DEFAULT CHARSET=utf8;

  • JDBC 连接url加上编码的参数
    jdbc:mysql://127.0.0.1:3306/mybatisdemouseUnicode=true&characterEncoding=utf8
